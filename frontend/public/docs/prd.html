1. Product Vision

ON24 Nexus is the everything app for marketers who care about engagement and data. It is not just an analytics tool — it is the intelligence layer that connects every webinar a team runs to actionable insights, reusable content, and smarter future planning. The system compounds in value over time: every webinar transcript, every AI-generated article, and every audience signal is retained and used to inform the next answer, the next piece of content, and the next event.

Nexus covers the full marketer lifecycle: understand what happened (analytics), answer how to do things (concierge), create new content tuned to your brand (content creation), and plan what to do next (forward-looking strategy) — all through a single conversational interface.

2. Feature Areas

2.1 Conversational Analytics (Data Agent)

Users ask questions in plain English and receive instant, visualized answers drawn directly from the ON24 master database.

Show me attendance trends for the past year → Renders a line chart of monthly registrants and attendees (12-month window)
Which companies attended the last webinar? → Returns a ranked bar chart of top companies with attendee and engagement data
How did "Next Gen AI Summit" do? → Shows an event KPI card: registrants, attendees, conversion rate, avg engagement
Show me poll results for the last event → Renders interactive poll cards with bar charts showing response distributions
Show performance by campaign tags → Aggregates KPIs per campaign tag (Webinars, EMEA, APAC, Demo, etc.) with attendance and engagement
Show funnel breakdown → Bar chart of events grouped by funnel stage (#stageAwareness, #stageConsideration)

2.1.1 Trends Sub-Menu

A dedicated "Trends" chip on the home screen opens a sub-menu with 7 pre-built analytics views:

  • Attendance over time
  • Registrations over time
  • Engagement scores over time
  • Show funnel (bar chart by funnel stage)
  • Show campaigns (pie chart by campaign)
  • Performance by tags
  • Top events by engagement

2.1.2 File Uploads

Users can attach PDF or image files (up to 10MB) via a paperclip button in the chat input. PDFs are text-extracted via PyMuPDF and injected as LLM context. Files are stored in data/uploads/ with automatic daily cleanup (files older than 24 hours are removed).

2.2 Concierge (Platform How-To)

Users ask how to use ON24 and receive immediate, self-contained answers from a curated knowledge base of 637 help articles and 71 ON24 REST API v2 endpoint descriptions — no links to external documentation required for common questions. API/endpoint/integration questions are routed to the knowledge base rather than deflected as out-of-scope.

How do I configure a registration page? → Delivers a complete step-by-step answer covering themes, fields, compliance, and next steps
How do I set up polls? → Explains poll creation in Elite Studio, widget placement, and where to review results

2.3 Event Calendar

An Outlook-style calendar modal showing all events for the current month with color-coded event types, KPI overlays, and a side panel for full event detail including AI-generated content.

  • Month / Week / Day view switching
  • Side panel: registrants, attendees, conversion, avg engagement, poll/survey/resource counts
  • AI-ACE content tile: Key Takeaways (sectioned), Blog, eBook, FAQ, Follow-up Email, etc.
  • Double-click to send event to chat for deeper analysis

2.4 Content Intelligence & Creation (Content Agent)

The content agent does two distinct jobs: it analyzes historical performance to surface strategy insights, and it creates new content tuned to the team's established brand voice.

Strategy analysis:

  • Top-performing topics by engagement score
  • Optimal scheduling patterns by day/time
  • Topic gap analysis against past event history
  • Propose a future webinar schedule with titles, data-driven justification, and 3-5 campaign themes per calendar

Content creation (brand-voice tuned):

  • Brand voice is built automatically from all AI-ACE content in the account's video_library — no manual setup needed
  • Every piece of new content is silently informed by the last 5 articles of the same type, ensuring consistency
  • Supported types: webinar scripts, blog posts, social media posts, eBooks
  • Content created here is grounded in webinar transcripts and audience insights, not generic prompting

2.5 Content Exploration (AI-ACE Library)

All AI-generated content from past webinars (Key Takeaways, blog posts, FAQs, follow-up emails, social media posts, eBooks) is browseable and explorable through chat. Users can ask to see, summarize, or repurpose any article.

Show me the most recent blog posts → Lists AI-generated blog articles from video_library, most recent first, with event context
Propose a 6-month webinar schedule → Returns titled webinar proposals with engagement-justified rationale for each, drawing on transcript and poll data to identify what topics audiences want next

2.6 Event Management (Admin Agent)

Enables event creation, registration management, and updates via chat with explicit confirmation before any write operation.

  • Create new events with title, date, description, type
  • Add or remove registrants
  • Update event details
  • All write operations require user confirmation dialog before execution

3. Agent Architecture

Data Agent — Handles all analytics queries. Has 12 tools covering events, KPIs, attendance trends, audience companies, poll results, Q&A, resources, tag/campaign queries, and chart generation.
Concierge — Answers platform how-to questions using a Postgres-backed knowledge base (OpenAI embeddings, cosine similarity). Delivers complete answers without external links.
Content Agent — Provides content strategy recommendations via topic analysis, scheduling patterns, and performance comparison tools.
Admin Agent — Executes ON24 write operations (create event, manage registrations) with mandatory user confirmation.

5. User Stories

Analytics

  • As a demand gen manager, I want to see last month's top attending companies so I can prioritize account outreach.
  • As a program manager, I want attendance trends for the past year so I can identify seasonal patterns.
  • As a content strategist, I want to see poll results for a specific event so I can understand audience sentiment.

Platform Help

  • As a marketing ops user, I want to ask how to configure a registration page and get an immediate answer without leaving the platform.
  • As a new ON24 user, I want to understand how polls work before my first event.

Calendar

  • As a program manager, I want to see all events for the current month with at-a-glance KPIs.
  • As a content team member, I want to view AI-generated key takeaways from a past event directly in the calendar.

7. Out of Scope

  • Per-speaker performance metrics (not tracked at speaker level in ON24 DB)
  • Geographic / regional audience breakdown
  • Revenue, ROI, or financial attribution
  • Email campaign metrics
  • Real-time live event monitoring
  • Multi-tenant / multi-client switching (planned; not yet implemented)
