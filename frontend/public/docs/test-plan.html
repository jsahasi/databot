How to Profile

python -m pytest tests/test_response_times.py -v

Requires running backend (docker compose up). Results saved to tests/results/response_time_results.json. Run after any performance-related change to track impact.

How to Run

Unit & Integration Tests

python -m pytest backend/tests/ -v

Runs from project root. Requires pip install -e ".[dev]" in backend/.

Prompt Regression Tests

python -m pytest tests/test_chat_prompts.py -v

Requires running backend (docker compose up). Uses per-test session IDs with 2s delay between tests.

Benchmarks (not in CI)

python backend/tests/benchmark_read_apis.py

3-way comparison: REST API vs MCP server vs direct DB queries. Requires all services running.
