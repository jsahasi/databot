Navigation Depth Tests — Tier 1 / 2 / 3

Every user journey from the home page is covered to at least three levels. Tier 1 = home page tile. Tier 2 = first sub-menu or modal. Tier 3 = agent response or external destination reached from Tier 2.

Performance SLA: any Tier-2→3 transition that calls an LLM or DB must complete in <5 s (cached) or <15 s (cold). Items that exceed 5 s on first load are pre-warmed in the startup prefetch. ✅ = pre-warmed | 🔶 = LLM required, not pre-warmed

How to Run

Unit & Integration Tests
